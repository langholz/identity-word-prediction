What is this?

This is an application that demonstrates identity-based text word and sentence prediction through language models created using Python 3.6.2, PyTorch 0.2.0_3 and Flask.

Who is it for?

This application and its resources have been created for the University of Illinois at Urbana-Champaign Master of Computer Science in Data Science CS410 Text Information Systems class. However, it is for anyone and everyone who like to tinker and play with technology.

How does it work?

At the root of the application two character-level neural language models (one for Homer Simpson and the second for Sherlock Holmes) were created using a Recurrent Neural Network with Long short-term memory (RNN-LSTM) layers.

First, a two layer generic identity model was trained using the Cornell University Movie Dialog Corpus. Then, the identity models were constructed by copying the generic identity model layers, freezing them and adding two more identity layers after them. Since the first two layers of the model (the generic identity layers) were frozen, upon training, the last two layers were updated to encapsulate the specific identity of the character.

What future improvements would you perform?

Model improvements:

  • Experiment with implementing and using beam search to improve the generation of words, sentences and reduce typos.
  • Increase the length of the training sequences so that it spans a paragraph.
  • Further optimize training and model so that we can learn faster with fewer amount of data by possibly trying out using word-level language model.
  • Allow further adapting the identity model as the user sees fit.
  • From the same model, allow for generating text from the generic identity layers directly without taking into account the character identity layers.

Application improvements:

  • Create a mobile device application that leverages these models.
  • Demonstrate that the model also performs word completion.
